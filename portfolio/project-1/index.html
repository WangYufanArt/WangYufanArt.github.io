Subverters1

有为终成就于无为.

这是一组3D打印和针织服装相结合的设计尝试。

Project Introduction

Subverters是一种状态，也是一种形式。

从意识形态和象征意义的角度说，它是个体欲望和社会规则之间的博弈。选用的材料是3D打印和针织。3D打印的部分是坚硬的，代表社会规则，是构建速食社会的工业生产方式下科技程序的冰冷感，以及工业社会对人的束缚、物化；而个体存在的本真状态自由如针织，呈现出柔软舒服而贴合的自然性，代表了“以人为本”针织工艺柔软坚韧的温度。

这个设计表现了“人”的存在状态。它即是每个人内心自我的真实表达，也呈现出个人在成长的过程中对社会制度与规则的服从。它可以是愚者的天真，或在规则中迷失的自我。最后，旧的自我终归死去，愿新的自我为颠覆者冲破枷锁，在社会构建中回归本心，实现真心。

这是一个“有为”终成就于“无为”的状态。

未来的服装应该是利用新技术、新思想来颠覆现实的新形态和新方式，表现人和机器和谐共处的共存状态。3D打印和针织两种设计手法相结合的方式就是试图在极软的针织和极硬的3D打印中找到平衡，探讨软硬结构之间的界限关系。而在针织服装中加入3D打印颠覆了针织服装本身的造型形态，3D打印和针织这两种表达方式都是从无到有的增量设计，在设计过程中实现了零剪裁和零浪费。
