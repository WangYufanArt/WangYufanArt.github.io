Subverters1

有为终成就于无为. 这是一组3D打印和针织服装相结合的设计尝试。 Project Introduction Subverters是一种状态，也是一种形式。 从意识形态和象征意义的角度说，它是个体欲望和社会规则之间的博弈。选用的材料是3D打印和针织。3D打印的部分是坚硬的，代表社会规则，是构建速食社会的工业生产方式下科技程序的冰冷感，以及工业社会对人的束缚、物化；而个体存在的本真状态自由如针织，呈现出柔软舒服而贴合的自然性，代表了“以人为本”针织工艺柔软坚韧的温度。 这个设计表现了“人”的存在状态。它即是每个人内心自我的真实表达，也呈现出个人在成长的过程中对社会制度与规则的服从。它可以是愚者的天真，或在规则中迷失的自我。最后，旧的自我终归死去，愿新的自我为颠覆者冲破枷锁，在社会构建中回归本心，实现真心。 这是一个“有为”终成就于“无为”的状态。 未来的服装应该是利用新技术、新思想来颠覆现实的新形态和新方式，表现人和机器和谐共处的共存状态。3D打印和针织两种设计手法相结合的方式就是试图在极软的针织和极硬的3D打印中找到平衡，探讨软硬结构之间的界限关系。而在针织服装中加入3D打印颠覆了针织服装本身的造型形态，3D打印和针织这两种表达方式都是从无到有的增量设计，在设计过程中实现了零剪裁和零浪费。 More About Project

生来就是两面派

本系列作品是一组关于无性别（Unisex）服装的设计 时尚就是一场轮回——我们所为之疯狂的潮流往往要追溯于那些历史上的经典，而每一次创意的激发和创新的诞生总会或多或少的带着过去的痕迹。 选取六位模特分别反串扮演最具无性别主义特征的六位文化偶像：奥斯卡·王尔德，弗吉尼亚·伍尔芙，弗里达·卡罗，西蒙娜·波伏娃，艾伦·图灵，大卫·鲍威。他们身上的六套look，代表的六种品质，也是他们最有代表性、最经典的时尚符号。 （一）奥斯卡·王尔德 Oscar Wilde 不管对女士还是男士，风衣都是秋日时尚的经典品类，不仅防风，而且拉风。历史上也有一位才华横溢又穿着拉风的文豪——奥斯卡·王尔德。 作为十九世纪英伦上流社会“花花公子(dandy)”的领军代表，王尔德爱穿华丽的天鹅绒外套和绸缎长裤，精致得一丝不苟。他的时尚品位体现着美的哲学：他崇尚和歌颂一切美，认为美不再是女性专利，拥有浓重的纳喀索斯情结。其中性化的唯美主义做派影响了无数人。 “自恋是一生的浪漫的开始。” （二）西蒙娜·波伏娃 Simone Beauvoir 但波伏娃和性感没什么关系，她从不把自己打扮成取悦男性的玩偶。她总穿肥大和严肃的外衣，梳着光亮的发髻，带着女强人的味道。站在萨特和他们年轻漂亮的情人中间，波伏娃总显出一种禁欲气息的凛冽和干练。
